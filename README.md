# Administrative vs standard user privilege:-
1.Administrative User (Admin):-
Users with full control over the system.
What they can do:
Install or remove software
Change system settings
Create, delete, or modify user accounts
Access all files
Configure security settings
Risk:
If hacked, the attacker gets full system control

2. Standard User:-
Users with limited access for daily tasks.
What they can do:
Use installed applications
Access personal files
Change basic settings (like wallpaper)
Browse the internet
Limitations:
Cannot install software
Cannot change critical system settings
Cannot manage other users
